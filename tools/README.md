# Tools and Resources

Comprehensive list of tools for retro game development.

## Development Tools

### Game Engines

#### Unity
- **Website:** [unity.com](https://unity.com/)
- **License:** Free (Personal), Paid (Pro)
- **Platform:** Windows, macOS, Linux
- **Best for:** Beginners, 2D/3D games
- **Learning:** [Unity Learn](https://learn.unity.com/)

#### Godot
- **Website:** [godotengine.org](https://godotengine.org/)
- **License:** Free, Open Source (MIT)
- **Platform:** Windows, macOS, Linux
- **Best for:** Open source projects, 2D games
- **Learning:** [Godot Docs](https://docs.godotengine.org/)

#### GameMaker
- **Website:** [gamemaker.io](https://gamemaker.io/)
- **License:** Paid (Free trial)
- **Platform:** Windows, macOS
- **Best for:** 2D games, no coding option
- **Learning:** Built-in tutorials

### Code Frameworks

#### SDL2 (Simple DirectMedia Layer)
- **Website:** [libsdl.org](https://www.libsdl.org/)
- **License:** Free, Open Source (Zlib)
- **Language:** C/C++
- **Best for:** Maximum control, low-level
- **Tutorial:** [Lazy Foo' Productions](https://lazyfoo.net/tutorials/SDL/)

#### SFML (Simple and Fast Multimedia Library)
- **Website:** [sfml-dev.org](https://www.sfml-dev.org/)
- **License:** Free, Open Source (Zlib)
- **Language:** C++
- **Best for:** C++ developers
- **Tutorial:** [Official tutorials](https://www.sfml-dev.org/tutorials/)

#### Raylib
- **Website:** [raylib.com](https://www.raylib.com/)
- **License:** Free, Open Source (Zlib)
- **Language:** C (bindings for many languages)
- **Best for:** Learning, prototyping
- **Tutorial:** [Examples](https://www.raylib.com/examples.html)

#### MonoGame
- **Website:** [monogame.net](https://www.monogame.net/)
- **License:** Free, Open Source
- **Language:** C#
- **Best for:** XNA developers
- **Tutorial:** [Documentation](https://docs.monogame.net/)

- **Website:** [piskelapp.com](https://www.piskelapp.com/)

## Audio Tools

### Music Creation

#### BeepBox (Easiest)
- **Website:** [beepbox.co](https://www.beepbox.co/)
- **License:** Free, Open Source
- **Platform:** Browser
- **Best for:** Instant chiptune music
- **Export:** WAV, MIDI, JSON

#### FamiStudio
- **Website:** [famistudio.org](https://famistudio.org/)
- **License:** Free, Open Source
- **Platform:** Windows, macOS, Linux
- **Best for:** NES-style music
- **Export:** NSF, WAV, various formats

#### MilkyTracker
- **Website:** [milkytracker.org](https://milkytracker.org/)
- **License:** Free, Open Source
- **Platform:** Windows, macOS, Linux
- **Best for:** MOD/XM music (Amiga-style)
- **Format:** Tracker interface

#### LMMS
- **Website:** [lmms.io](https://lmms.io/)
- **License:** Free, Open Source
- **Platform:** Windows, macOS, Linux
- **Best for:** Full DAW, chiptune
- **Features:** VST support, built-in instruments

#### Bosca Ceoil
- **Website:** [terrycavanagh.itch.io/bosca-ceoil](https://terrycavanagh.itch.io/bosca-ceoil)
- **License:** Free
- **Platform:** Windows, macOS, Linux, Browser
- **Best for:** Simple music creation
- **Export:** WAV, MID

### Sound Effects

#### Bfxr/Sfxr
- **Website:** [bfxr.net](https://www.bfxr.net/)
- **License:** Free, Open Source
- **Platform:** Browser, Desktop
- **Best for:** Retro SFX generation
- **Features:** Randomize, presets

#### ChipTone
- **Website:** [sfbgames.itch.io/chiptone](https://sfbgames.itch.io/chiptone)
- **License:** Free
- **Platform:** Browser
- **Best for:** Detailed chiptune SFX
- **Features:** Visual waveform editing

#### Audacity
- **Website:** [audacityteam.org](https://www.audacityteam.org/)
- **License:** Free, Open Source
- **Platform:** Windows, macOS, Linux
- **Best for:** Audio editing, effects
- **Features:** Multi-track, effects, export

### C64 Music Tools

#### GoatTracker
- **Website:** [csdb.dk](https://csdb.dk/release/?id=197888)
- **License:** Free, Open Source
- **Platform:** Windows, Linux
- **Best for:** Authentic C64 SID music
- **Export:** SID files

#### SID-Wizard
- **Website:** [csdb.dk](https://csdb.dk/release/?id=156144)

- **Platform:** C64, emulator

- **Website:** [sourceforge.net/projects/acme-crossass](https://sourceforge.net/projects/acme-crossass/)

- **Platform:** Cross-platform
- **Best for:** 6502 Assembly
- **Features:** Macros, multiple passes

#### Kick Assembler
- **Website:** [theweb.dk/KickAssembler](http://theweb.dk/KickAssembler/)

- **Platform:** Java (cross-platform)
- **Best for:** Advanced C64 assembly
