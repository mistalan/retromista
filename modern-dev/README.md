# Modern Game Development Frameworks

Build retro-style games using modern tools and engines for cross-platform distribution.

## Recommended Engines

### 1. Unity (Recommended for Beginners)

**Why Unity?**
- Easy to learn
- Huge community and resources
- Free for personal use
- Cross-platform export

**Setup:**
1. Download Unity Hub from [https://unity.com/](https://unity.com/)
2. Install Unity LTS version (currently 2022.3 LTS)
3. Choose "2D" template for retro games

**Getting Started:**
See [Unity Setup Guide](unity/README.md)

### 2. Godot (Open Source Alternative)

**Why Godot?**
- Completely free and open source
- Lightweight and fast
- Good 2D engine
- No licensing fees

**Setup:**
1. Download Godot from [https://godotengine.org/](https://godotengine.org/)
2. No installation needed - just run
3. Great for 2D pixel art games

**Getting Started:**
See [Godot Setup Guide](godot/README.md)

### 3. Visual Studio with C++ / C# (For Advanced Users)

**Why Visual Studio?**
- Full control over everything
- Maximum performance
- Professional development environment
- Works with various frameworks (SDL, SFML, MonoGame)

**Setup:**
1. Download Visual Studio Community from [https://visualstudio.microsoft.com/](https://visualstudio.microsoft.com/)
2. Install C++ or C# workload
3. Choose a game framework (see below)

**Recommended Frameworks:**
- **SDL2** (C/C++) - Low level, maximum control
- **SFML** (C++) - Simple, multimedia library
- **MonoGame** (C#) - XNA framework successor
- **Raylib** (C/C++) - Simple and beginner-friendly

**Getting Started:**
See [Visual Studio Setup Guide](visual-studio/README.md)

## Framework Comparison

| Engine | Language | Difficulty | Best For |
|--------|----------|------------|----------|
| Unity | C# | Easy | Beginners, rapid prototyping |
| Godot | GDScript/C# | Easy | Open source enthusiasts |
| SDL2 | C/C++ | Hard | Maximum control |
| SFML | C++ | Medium | C++ developers |
| MonoGame | C# | Medium | XNA developers |
| Raylib | C/C++ | Easy | Learning game programming |

## Quick Start Recommendation

**New to programming?** → Start with **Unity**
- Most tutorials available
- Easiest to get something working
- Visual editor helps understanding

**Know programming basics?** → Try **Godot**
- Lightweight and free
- Good scripting language
- Active community

**Experienced programmer?** → Use **Visual Studio + Framework**
- Maximum flexibility
- Best performance
- Professional tools
